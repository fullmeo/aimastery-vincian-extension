# Roadmap Windsurf - Plan Mode

Application web interactive pour la gestion de projet selon la méthodologie Windsurf en mode Plan.

## Structure du Projet

```
windsurf-plan-mode/
├── index.html         # Page principale
├── css/
│   └── style.css     # Styles principaux
├── js/
│   └── main.js       # Fonctionnalités interactives
├── assets/            # Ressources diverses
├── img/               # Images
└── docs/              # Documentation
```

## Fonctionnalités

- Vue d'ensemble du projet
- Phases stratégiques détaillées
- Outils clés
- Bonnes pratiques
- Tableau de bord des métriques

## Installation

1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur web

## Développement

- HTML5 sémantique
- CSS3 avec variables CSS
- JavaScript vanilla (ES6+)
- Design responsive
